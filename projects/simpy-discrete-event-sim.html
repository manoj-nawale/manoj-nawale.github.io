Discrete-Event Simulation of Production Flow (SimPy)

A circular-production digital twin that models both forward manufacturing flow and reverse logistics using SimPy. The simulation tracks key KPIs such as throughput, inventory dynamics, and cost under different operating policies. Results are explored through a lightweight dashboard for scenario comparison and decision support.

How to run

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python -m apps.run_sim
python -m apps.dashboard
python -m apps.run_sim --config configs/scenarios/returns_high.yaml
